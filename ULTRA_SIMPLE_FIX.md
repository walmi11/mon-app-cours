## ÉTAPE 1: Cherche cette section sur Vercel

Regarde la page Vercel où tu ajoutes les variables.

**Cherche du texte qui dit:**
```
"or paste the .env contents above"

"Paste environment variables"

```

1. **Sélectionne TOUT le texte ci-dessus** (Ctrl+A)
2. **Copie** (Ctrl+C)
3. **Va à Vercel**
4. **Clique dans le grand champ**
5. **Colle** (Ctrl+V)

## ÉTAPE 3: Clique un bouton
